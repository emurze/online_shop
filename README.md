### How to contribute?

```git clone git@github.com:emurze/online_shop.git```

```git checkout -b "feature/new_feature"```

> New Code added

```git add .```

```git commit -m "new code"```

```git push -u origin "feature/new_feature"```

Pull changes from github to your local project

```git pull```
